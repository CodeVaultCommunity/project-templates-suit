# Project Starters Hub

A collection of reusable, production‑ready starter configurations for various types of software projects.  
Each starter acts as a scaffold containing best practices, tooling setup, and recommended workflows.

## Contents
- TypeScript Starter with ESLint, Sonar, Commitlint, Husky, and GitHub Workflows
- (More starters coming soon…)

## Purpose
This repository centralizes common project setups so they can be cloned, imported, or used as Git submodules across new projects.

## How to Use
1. Navigate to the starter folder you want.
2. Add it as a submodule or copy it into your new project.
3. Follow the instructions in the starter’s individual README.
